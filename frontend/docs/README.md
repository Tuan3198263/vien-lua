1. Đọc [Tổng Quan](./frontend-guides/01-overview.md) để hiểu công nghệ và cấu trúc
2. Xem [Ưu Tiên Ant Design](./frontend-guides/02-ant-design-priority.md) để biết cách dùng Ant Design
3. Tham khảo [Components](./frontend-guides/03-components.md) khi tạo component mới
4. Luôn check [Checklist](./frontend-guides/08-checklist.md) trước khi commit
